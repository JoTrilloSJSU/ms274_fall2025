# MS 274 Fall 2025
This repository contains files for any homework and project submissions for an Ocean Modeling Class taught at Moss Landing Marine Labs for Fall 2025. A summary of the content in each directory is provided below:
## Homework
This directory contains weekly homework submisions and early verions of code made for creating initial, boundary, and external forcing conditions for the final project.
## Project (ocean mixing in SLO Bay)
This directory contains model code, visualizations, and any notebooks used for model initialization and analysis.
